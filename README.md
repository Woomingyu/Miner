# Miner

독학 당시 진행한 개인 프로젝트 입니다.  
여러 기능의 구현과, 검색이나 강의 등에서 찾은 코드들을 활용하거나, 변형하여  
문제없이 작동할 수 있도록 개발하는 것에 집중하였습니다.  


***
